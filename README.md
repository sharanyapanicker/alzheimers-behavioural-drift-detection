# Alzheimer's Behavioural Drift Detection

MSc Artificial Intelligence with Business Strategy Dissertation
Aston University

## Overview

This project investigates the generation and validation of synthetic smart-home sensor data for detecting behavioural drift associated with early-stage Alzheimer's disease.

The study extends the SHARON (Simulator of Human Activities, ROutines and Needs) framework to simulate Activities of Daily Living (ADLs) and progressive dementia-related behavioural changes.

## Key Features

- Synthetic smart-home sensor data generation
- Behavioural drift simulation
- Validation against CASAS benchmark dataset
- Entropy and Jaccard similarity analysis
- Anomaly detection using classical and deep learning models

## Models Evaluated

- Isolation Forest
- One-Class SVM
- Autoencoder
- LSTM
